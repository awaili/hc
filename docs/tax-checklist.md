# 本草链 HerbChain · 税务合规检查清单（中药材大货）

> 中药材大货贸易是税务高风险行业（农产品收购发票、虚开高发）。本清单用于需求调研与系统设计时的合规底线参考，并与访谈/摘要的"税务合规"分类对应。

## 1. 进项——收购端
- [ ] 向**自产农户**收购：是否开具农产品收购发票/收购凭证？是否核实种植户身份（身份证）、留存**自产证明**？
- [ ] 收购金额是否与农户自产合理产量匹配（防超额虚开）？
- [ ] 向**经营户/合作社/初加工户**收购：是否取得对方开具的发票（而非自开收购票）？
- [ ] 农产品进项按 **9%** 计算抵扣，是否准确入账、抵扣凭证是否齐全？
- [ ] **无票采购**部分：是否单独标记？（可进成本但**不得抵扣进项**，抬高实缴税负）

## 2. 进项——加工与费用端
- [ ] 外包加工费、辅料、包装、电费、运费是否取得合规发票？
- [ ] 自有加工的人工成本：工资社保是否合规？外包给个人的加工费是否代扣个税？

## 3. 销项——销售端
- [ ] 对药企/饮片厂是否开具 **13% 增值税专用发票**？普票/专票场景是否区分？
- [ ] 开票时点与收入确认是否一致？账期开票节奏怎么管？
- [ ] 红冲、退货、折让的开票处理流程？
- [ ] 客户开票资质（发票抬头、税号、专票资质）是否建档核验？

## 4. 三流一致（防虚开，重点）
- [ ] **货流、资金流、发票流**是否一致（同买同卖同账户同金额）？
- [ ] 是否存在**个人账户收取对公货款**的情况？（虚开高风险，需纳入系统监控并提示）
- [ ] 资金回流路径是否可追溯？

## 5. 其他税种
- [ ] 购销合同**印花税**是否计提申报？
- [ ] 付给个人的款项（农户货款、散工、代购佣金、装卸）是否**代扣代缴个税**？
- [ ] 附加税（城建/教育/地方教育附加）随增值税计提？

## 6. 药监 / GSP / 行业合规（非税但关联）
- [ ] 客户与上游资质审核（药品生产/经营许可证、GSP）是否建档并在效期内？
- [ ] 批次追溯：每批货产地→加工→销售可逆向追查。
- [ ] COA / 质检报告随货；含量、农残、重金属、黄曲霉、SO2 达药典。
- [ ] 来源合法：防伪劣、防掺伪、防野生保护物种（如部分野生药材受限）。

## 7. 与分润口径的衔接（决策项 #1）
- 若分润按"净利"，**税负**（实缴增值税+附加+所得税）应否计入成本？
  - 进项票缺口越大 → 实缴税负越高 → 净利越低 → 直接影响合伙人分润，须在系统里量化可见。
- 系统应能按批次/订单计算"含税成本"与"税后净利"，避免分润时漏算税负。

## 8. 系统需落地的合规功能（对应需求设计）
- 收购单强制登记：种植户身份、自产证明附件、收购发票号、是否可抵扣。
- 无票采购标记 + 税负影响测算。
- 销售开票管理：开票申请、专票/普票、红冲、客户资质校验。
- 三流一致性校验：订单/发货/收款/开票四单匹配预警。
- 个税代扣台账（付个人款项）。
- 资质效期预警（客户/上游许可证）。
- 批次追溯链 + COA 出具。

## 9. 合规节税 / 税收筹划方向（守红线前提下，需税务师/会计确认）
- [ ] **用足农产品免税政策**：向自产农户/合作社采购，凭证齐全 → 合规且得合法发票/抵扣；合作社销售成员自产农产品免增值税。
- [ ] **纳税人身份测算**：小规模（1%/3%）vs 一般纳税人（13%可抵进项）。按客户结构（药企要专票 vs 市场散户不要专票）测算两种身份实缴税负，择优。
- [ ] **进项最大化**：加工费/辅料/运费/电费/包装尽量取得合规发票，降低无票采购比例 → 降低实缴增值税。
- [ ] **业务环节拆分**：农产品初加工所得符合《农产品初加工所得优惠目录》享企业所得税免/减；初加工单独核算。
- [ ] **小微/所得税优惠**：符合小微条件享所得税减免；个体户/个独经营所得、合理薪酬与利润分配结构。
- [ ] **时间性安排**：收入确认与成本结转时点、存货跌价与损耗合规确认，平滑利润。
- [ ] **系统量化**：按批次算"含税成本/实缴税负/税后净利"，对比采购与开票方式，选合规且税负低方案。

## 10. 红线（绝不可碰，系统须监控拦截）
- ❌ 虚开发票（无货虚开、买票卖票）、取得虚开进项抵扣
- ❌ 隐瞒收入、两套账
- ❌ 个人卡收对公货款、三流不一致、资金回流可疑
- ❌ 超过农户自产合理量开具收购发票
- ❌ 利用假自产证明、虚假农户身份套取收购发票抵扣
